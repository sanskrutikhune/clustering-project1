Unsupervised Learning: Clustering

Grouping data points based on similarity

What is Clustering?

Clustering is a technique in data mining and machine learning that groups a set of objects in such a way that objects in the same group (called a cluster) are more similar to each other than to those in other groups.

1. Centroid-based

Clusters are represented by a central vector. It is efficient but sensitive to outliers.


Example: K-Means Clustering.

2. Density-based

Locates regions of high density and separates them from regions of low density.


Example: DBSCAN, OPTICS.

3. Distribution-based

Assumes data is composed of distributions like Gaussian distributions.


Example: Expectation-Maximization (EM).

4. Hierarchical

Builds a tree of clusters (Dendrogram).


Example: Birch, CURE.
