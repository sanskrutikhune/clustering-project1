What is Clustering?

Clustering is an unsupervised machine learning technique used to group similar data points together. It helps in discovering patterns and structures in data without labeled outputs.

1. K-Means Clustering (Centroid-Based)

K-Means divides data into K clusters using centroids.

Advantages:
  • Simple and fast
  • Works well for large datasets
Disadvantages:
  • Sensitive to outliers
  • Need to choose K value
Applications:
  • Customer Segmentation
  • Market Analysis

2. Density-Based Clustering (DBSCAN)

Forms clusters based on dense regions of data.

Advantages:
  • Can detect irregular shapes
  • Handles noise well
Disadvantages:
  • Hard to select parameters
Applications:
  • Geographical data analysis
  • Fraud detection

3. Distribution-Based Clustering (Gaussian)

Assumes data follows a probability distribution.

Advantages:
  • Good for probabilistic models
Disadvantages:
  • Assumes data distribution shape
Applications:
  • Image processing
  • Scientific data analysis

4. Hierarchical Clustering

Builds clusters in a tree-like structure (dendrogram).

Advantages:
  • No need to specify cluster number initially
Disadvantages:
  • Computationally expensive
Applications:
  • Biology (Gene classification)
  • Document clustering
