What is Clustering?

Clustering is a machine learning technique that groups similar data points together. It is mainly used in Unsupervised Learning where no labeled data is provided.

1. Centroid-based

Clusters are represented by a central vector.

Example: K-Means

2. Density-based

Forms clusters based on dense regions.

Example: DBSCAN

3. Distribution-based

Assumes data follows a probability distribution.

Example: EM Algorithm

4. Hierarchical

Creates tree-like cluster structure.

Example: BIRCH

Advantages

  • Easy to understand and implement.
  • Helps in pattern recognition.
  • Useful for large datasets.
  • No labeled data required.

Disadvantages

  • Results depend on selected algorithm.
  • Sensitive to outliers (K-Means).
  • Choosing number of clusters is difficult.

Applications

  • Customer Segmentation
  • Image Processing
  • Market Analysis
  • Medical Data Analysis

Visualization

Clustering results are commonly visualized using scatter plots where different clusters are shown in different colors for better understanding.
